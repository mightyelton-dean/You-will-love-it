# For Goodness 👑🌸

A stunning appreciation website with a dramatic intro for Goodness.

## File Structure
```
index.html   ← Intro page (entry point — this loads first)
main.html    ← Main appreciation website
style.css    ← Styles for the main site
script.js    ← Animations for the main site
photo1.jpg   ← Goodness photo 1
photo2.jpg   ← Goodness photo 2
```

## Deploy on GitHub Pages
1. Create a new GitHub repository
2. Upload ALL 7 files to the **root** of the repo
3. Go to **Settings → Pages**
4. Set source to **Deploy from branch → main → / (root)**
5. Click **Save** 🎉

Your site will be live at: https://yourusername.github.io/repo-name
